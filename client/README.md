# Buildpath

- **template of:** vite + react + typescript
- **tailwindcss:** v3.x
- **project management:** [ClickUp](https://app.clickup.com/8445079/v/l/s/49664568)
- **mockups:** [Figma](https://www.figma.com/file/3HuZeyRnkf8yQrEQeBFLh4/Buildpath?node-id=0%3A1)

---

## Started

```bash
pnpm install
# or `yarn`
# or `npm install`
```

## Develop

```bash
pnpm dev
# or `yarn dev`
# or `npm run dev`
```


## Build

```bash
pnpm build
# or `yarn build`
# or `npm run build`
```
